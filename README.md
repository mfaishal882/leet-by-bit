## Install

1. Install latest golang version.
2. Update Go Module

```
// remove current go.mod
rm go.mod

// init new go module
go mod init <project-name>

// update package
go mod tidy
```

## Usage

go run <folder_name>/<file_name>

go run concate_array/concate_array.go
